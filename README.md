# WeatherNow
Weather App using HTML, CSS, and JavaScript that fetches real-time weather data from an API and displays temperature, humidity, and wind speed for any city.

WeatherNow is a simple and modern weather application that provides real-time weather information for any city in the world. It uses the OpenWeather API to fetch accurate weather data and displays it with a clean and dynamic user interface.

The app shows temperature, weather condition, wind speed, and humidity, along with a dynamic full-screen background that changes based on the current weather.
